WebAssembly (Wasm) 是一种可以在浏览器中运行的低级字节码格式。

## 什么是 WebAssembly？

WebAssembly 是一种编译目标，可以让 C、C++、Rust 等语言编写的代码在浏览器中运行。

### 主要特性

- **高性能**: 接近原生速度
- **安全**: 在沙箱环境中运行
- **可移植**: 跨平台支持
- **紧凑**: 二进制格式，体积小

## 使用场景

- 游戏引擎
- 图像/视频处理
- 音频处理
- CAD 应用
- 加密计算

## Hello World

使用 Rust 和 wasm-bindgen：

```rust
use wasm_bindgen::prelude::*;

#[wasm_bindgen]
pub fn greet(name: &str) -> String {
    format!("Hello, {}!", name)
}
```

在 JavaScript 中使用：

```javascript
import init, { greet } from './pkg/hello_wasm.js'

async function run() {
  await init()
  console.log(greet('WebAssembly'))
}

run()
```

## 性能对比

WebAssembly 在计算密集型任务中表现出色：

```javascript
// JavaScript 版本
function fibonacci(n) {
  if (n <= 1) return n
  return fibonacci(n - 1) + fibonacci(n - 2)
}

// WebAssembly 版本（从 Rust 编译）
import { fibonacci } from './wasm/fibonacci.js'

// WebAssembly 通常快 10-100 倍
```

## 总结

WebAssembly 为 Web 平台带来了新的可能性，让复杂的应用可以在浏览器中流畅运行。
